Distributing plugins on GitHub#

GitHub is a straight-forward way to share your QIIME 2 plugin. This tutorial will walk you through creating a QIIME 2 plugin from template, and then sharing it with installation instructions on GitHub.

This tutorial assumes that you have git and the GitHub command line interface installed on your computer. If you don‚Äôt have these installed, do that now using the GitHub command line interface installation instructions, which should install git if you don‚Äôt already have it installed.

Note

After completing those steps, return to this page.
